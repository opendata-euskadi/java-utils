Install
-------
- Unzip the apache24 folder to d:/http_server/apache24 (that is the ServerRoot in the config).
- In httpd.conf
        * Change ServerRoot to d:/http_server/apache24
        * Chand DocumentRoot to wherever the document root directory is going to be located

Start apache in a DOS box: 
        >httpd.exe

Windows Service (http://httpd.apache.org/docs/2.4/platform/windows.html#winsvc)
---------------
Install as a service with name: pci_httpd
        >httpd.exe -k install -n "pci_httpd" -f "d:\http_server\apache24\conf\httpd.conf"
Uninstall service:
        httpd -k uninstall -n "pci_httpd"

ApacheMonitor:
        Double click ApacheMonitor.exe, or put it in your Startup folder.
